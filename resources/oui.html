The Organizationally Unique Identifier (OUI) of EPFL is 00-60-d7.

Allocation of numbers within address spaces governed by that OUI can be requested from the email address at the bottom of this page.

Assignments currently in force are listed below. Code points have been assigned in the following number spaces:

  • RF hardware
  • ATM B-HLI Vendor-Specific Application Identifier

    RF Hardware

    The EPFL OUI can be used to identify experimental radio frequency hardware devices. The device number is composed of the EPFL OUI followed by 3 bytes, which are administered by the organization identified by the OUI.

    Identifier(s) Purpose
    00-17-02 RF Resonant Metalens

    ATM B-HLI Vendor-Specific Application Identifier

    ATM offers various mechanisms to select which application or protocol a connection is intended for. One of those mechanisms is to use the "Broadband High-Layer Information" (B-HLI), which can contain a "Vendor-Specific Application Identifier". Such an identifier is composed of a three-byte OUI, followed by four bytes which are administered by the organization identified by the OUI.

    Identifier(s) Purpose
    00-xx-xx-xx not available
    01-00-00-01 Arequipa
    01-00-00-02 ACP (ATM Connection Performance)
    01-00-00-03 ttcp_atm (TTCP with Linux ATM extensions)
